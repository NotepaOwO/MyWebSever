🚀 PoorWebServer

基于C++从零实现的工程级Web服务器

✅ 核心特性

  • 模块化架构设计（网络层、协议层、业务处理层）
  • 基于Socket API的TCP服务器（RAII资源管理）
  • HTTP/1.1协议解析（GET/POST方法、Header解析）
  • 线程池并发模型（避免每连接一线程问题）
  • OpenSSL HTTPS安全传输（TLS握手、证书验证）
  • CGI动态请求处理（fork/exec + 管道机制）

🎯 测试链接

  • 静态页面测试
  • POST表单测试
  • CGI脚本测试

💻 启动命令

# HTTP模式（默认端口8080） ./myhttp # HTTPS模式 ./myhttp --https --cert server.crt --key server.key --port 8443 # 自定义端口 ./myhttp --port 9000

🏗️ 设计模式

  • RAII模式：Socket、SSL资源自动管理
  • 工厂模式：HttpResponse创建方法
  • 线程池模式：高效并发处理
  • 代理模式：SSLWrapper封装OpenSSL

基于C++17 + OpenSSL + POSIX Socket实现
